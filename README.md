# Golden Pay

Mobile-first React fintech UI based on the provided screenshots.

## Run

```bash
npm install
npm run dev
```

Open the local Vite URL in your browser.

## Pages included

- Dashboard
- Bank Account form
- Security Deposit
- Withdrawal placeholder
- Transactions placeholder
- Support
- Sidebar menu

## Next backend options

You can connect this frontend to Node.js, Express, MongoDB, JWT auth, admin approval, deposit uploads, and withdrawal requests.
